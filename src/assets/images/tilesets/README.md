# Tilesets

This folder will contain tileset images.
Tileset images are made up of 16x16 tiles (image size is dynamic: if a file is 128x128 pixels each tile is 8x8 pixels, etc.)
Tiles are to be used as follows:
--------------------------------
[X,Y] - Function.
--------------------------------
[0,0] - Ceiling.
[0,1] - Wall.
[0,2] - Floor.
